# Dev Tools Platform

A micro-frontend developer tools platform built with React, TypeScript, and Vite.

## Features

- JSON Formatter
- Regex Tester
- JWT Decoder
- Base64 Encoder/Decoder
- Password Generator

## Architecture

Micro-frontend architecture using Module Federation.

## Getting Started

1. Install pnpm
2. Run: pnpm install
3. Run: pnpm dev

## Structure

- apps/shell - Main host application
- apps/* - Individual tool applications
- packages/* - Shared packages
